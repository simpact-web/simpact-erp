⬆ Upload Logos — SIMPACT

Uploadez les logos clients dans Supabase Storage et obtenez les URLs publiques à coller dans les fiches clients.

1 Créer le bucket "logos" dans Supabase

→ Allez sur Supabase Storage
→ Cliquez New bucket → nom : logos → cochez Public bucket → Create
→ Si le bucket existe déjà, passez à l'étape 2.

2 Sélectionnez les 3 fichiers PNG
Nom enregistré : attijari.png
